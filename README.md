# Agentic Applications with LlamaIndex and MCP

A hands-on project exploring how to build intelligent agentic AI applications using **LlamaIndex**, **Model Context Protocol (MCP)**, and **Weaviate**. This repository demonstrates how multiple AI agents can collaborate, retrieve knowledge from vector databases, interact with external tools, and execute complex workflows using enterprise-grade AI frameworks.

## Project Overview

This project focuses on building modern AI applications that go beyond single-prompt interactions by leveraging:

- Agentic AI architectures
- Tool-calling agents
- Retrieval-Augmented Generation (RAG)
- Model Context Protocol (MCP)
- Vector databases with Weaviate
- Multi-agent orchestration
- External data integrations

Throughout the project, you'll build increasingly capable AI systems that can retrieve information, reason over knowledge, delegate tasks between specialized agents, and integrate with external services.

## Features

- Build AI agents using **LlamaIndex**
- Implement tool calling and custom agent workflows
- Store and retrieve knowledge using **Weaviate Vector Database**
- Perform semantic search with Retrieval-Augmented Generation (RAG)
- Integrate external tools using **Model Context Protocol (MCP)**
- Develop role-based multi-agent systems
- Create scalable, enterprise-ready AI workflows

## Getting Started

Clone the repository:

```bash
git clone https://github.com/<your-username>/agentic-applications-llamaindex-mcp.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Configure your environment variables:

```bash
OPENAI_API_KEY=your_api_key
WEAVIATE_URL=your_weaviate_instance
WEAVIATE_API_KEY=your_api_key
```

Run the examples and explore each implementation to understand how agentic applications are designed and orchestrated.
